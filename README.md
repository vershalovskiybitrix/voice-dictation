# VoiceService — локальная диктовка речи (RU/EN)

Жмёшь горячую клавишу → говоришь → текст с пунктуацией появляется там, где курсор.
Всё работает **локально** (без облака) на базе [faster-whisper](https://github.com/SYSTRAN/faster-whisper).
Whisper сам расставляет пунктуацию и заглавные буквы и сам определяет язык (рус/англ).

## Возможности

- Два режима хоткея:
  - **PTT (правый Ctrl)** — держишь и говоришь, отпускаешь → вставка. Если во время
    удержания нажата **любая другая клавиша** (Ctrl+Home, Ctrl+C и т.п.) — ввод
    отменяется. Так обычные Ctrl-шорткаты не мешают диктовке.
  - **Toggle (Scroll Lock)** — нажал → пишет (горит лампочка Scroll Lock на клавиатуре),
    нажал снова → лампочка гаснет, текст вставлен.
- Вставка через буфер обмена (надёжно для кириллицы) с восстановлением прежнего буфера.
- Работает в системном трее, модель держится в памяти → мгновенная расшифровка.
- Звуковые сигналы старта/стопа, фильтр галлюцинаций (VAD + чёрный список фраз).
- Автофолбэк GPU → CPU, если CUDA недоступна.

## Установка (в т.ч. на другом компьютере)

Нужен **Python 3.10+** и (желательно) видеокарта NVIDIA с драйвером.

```powershell
git clone <repo-url> VoiceService
cd VoiceService
python application\install.py
```

Установщик создаёт окружение в `runtime/.venv` и ставит туда зависимости (включая
CUDA-библиотеки). Без NVIDIA-GPU приложение само работает на CPU (медленнее).

## Запуск

```powershell
# фоном, без консоли (иконка в трее):
application\run.bat

# или с логами в консоли:
runtime\.venv\Scripts\python.exe application\voiceservice.py
```

При первом запуске модель `medium` (~1.5 ГБ) скачивается автоматически (нужен интернет один раз).

## Использование

1. Поставь курсор в любое текстовое поле.
2. **PTT:** зажми **правый Ctrl**, говори, отпусти → текст вставится.
   (Если в это время нажмёшь ещё клавишу — ввод отменится, это норм для Ctrl-шорткатов.)
   **Toggle:** нажми **Scroll Lock** → загорится лампочка и пойдёт запись; нажми снова →
   лампочка погаснет и текст вставится.
3. Язык, паузу и выход — через меню иконки в трее.

> **Совет про язык.** По умолчанию язык определяется автоматически. На очень коротких
> фразах автоопределение иногда ошибается. Если диктуешь в основном на одном языке —
> выбери его явно в трее (Русский/English): и точнее, и чуть быстрее.

## Распознавание аудиофайлов (голосовые из мессенджеров и пр.)

Whisper распознаёт и готовые файлы — поддерживаются `.ogg/.opus` (Telegram/WhatsApp),
`.mp3`, `.m4a`, `.wav`, `.flac` и др. Три способа скормить файл:

1. **Трей → «Распознать аудиофайл…»** — выбираешь файл в диалоге. Оригинал не трогается.
2. **Папка-приёмник** — трей → «Открыть папку для распознавания», брось туда аудиофайл →
   распознаётся автоматически. После обработки файл уезжает в `inbox/done/`
   (хранятся последние ~20, потом подрезаются — диск не растёт).
3. **Командная строка / «Открыть с помощью»:**
   `python application\transcribe_file.py <файл>` — копирует файл в очередь (оригинал на месте).

Результат всегда копируется **в буфер обмена** и показывается в уведомлении.
Хочешь, чтобы он ещё и сразу вставлялся в активное поле — включи в трее
**«Вставлять результат файла в курсор»** (настройка запоминается).

**Telegram:** плагина внутри клиента нет, но удобно так — правый клик на голосовом →
«Сохранить как…» → выбрать папку-приёмник `runtime/inbox` → текст готов.

## Настройка (`runtime/config.json`)

Файл создаётся при первом запуске. Основные параметры:

| Параметр | Значение | Описание |
|---|---|---|
| `model` | `medium` | Размер модели: `small` / `medium` / `large-v3`. |
| `device` | `auto` | `auto` / `cuda` / `cpu`. |
| `language` | `auto` | `auto` / `ru` / `en` (можно менять в трее). |
| `ptt_key` | `ctrl_r` | Клавиша «зажать-и-говорить» (отмена при нажатии другой клавиши). |
| `toggle_key` | `scroll_lock` | Клавиша режима вкл/выкл. |
| `ptt_beep_delay` | `0.25` | Сек: задержка сигнала «пишу», чтобы не пищать на Ctrl-шорткаты. |
| `insert_method` | `clipboard` | `clipboard` (через буфер) или `type` (печать по символу). |
| `file_insert_at_cursor` | `false` | Вставлять результат из файла в курсор (помимо буфера). |
| `inbox_keep_processed` | `20` | Сколько обработанных файлов хранить в `inbox/done` (0 — удалять сразу). |
| `initial_prompt` | `…` | Словарь частых терминов/имён для точности (GitHub, Claude и т.п.). |
| `hallucination_blacklist` | список | Фразы-галлюцинации, которые отбрасываются. |

Имена клавиш — как в pynput: `ctrl_r`, `ctrl_l`, `scroll_lock`, `shift_r`, `space`,
`f9`, `caps_lock`, `pause`; обычные клавиши — сам символ (`a`, `1`).

## Что НЕ делается (и почему)

- Нет голосовых команд-знаков препинания — Whisper сам ставит пунктуацию.
- Нет постоянного прослушивания — запись только по хоткею (не жжёт ресурсы, не ловит обрывки).
- Нет облака, стриминга, диаризации — вне задачи диктовки.
